# Petunjuk

## Instalasi applikasi

hapus file package-lock.json

jalankan perintah `npm install`

## Memulai applikasi

jalankan perintah `npm start`

Buka [http://localhost:3000](http://localhost:3000) Untuk melihat aplikasi.
